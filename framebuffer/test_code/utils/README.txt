Directory 'FBdev' contains the 'fbtest' test suite from the linux fbdev project (http://sourceforge.net/projects/linux-fbdev/).

Directory 'internal' contains programs written within TI.

To build the tests, run the make.sh scripts as follows:
	# cd FBdev/utilities/fbtest/
	# ./make.sh
	# cd -
	# cd internal
	# ./make.sh
